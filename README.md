# Java-Music-Player
A command line Java program that can play music in .wav file format and allows basic controls over songs

---

## Features
- Music playing using command line input (7 choices)
- Pausing/continuing/restarting a song
- Changing a song
- Time control (skipping forwards or backwards)
- Audio control (increasing or decreasing volume)
  
---

## Requirements
Java 17.0+ installed

---

## Installation
1. Clone the repository:<br>
   https://github.com/Ofek-Hodis/Java-Music-Player <br>
   In terminal type "cd your/location/Music-Player" <br>
2. Run the program:<br>
   In terminal: "java -jar music-player.jar"
